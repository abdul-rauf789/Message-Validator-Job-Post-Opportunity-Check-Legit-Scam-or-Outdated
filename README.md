# Message-Validator-Job-Post-Opportunity-Check-Legit-Scam-or-Outdated
Built a small pipeline that takes a raw WhatsApp-style message (nothing pre-filtered) and runs it through a funnel:  1) Decide if it’s a job post at all  2) If yes, decide if it’s a real opportunity  3) If it passes, classify it as legit, scam-flavored, or outdated
